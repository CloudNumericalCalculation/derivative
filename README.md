# derivative
简单微分计算

1. 公式输入请不要换行
2. 乘号（`*`）请不要省略
3. `a~z`（小写）共26个字母可作为未知数使用
4. `A~Z`（大写）共26个字母可作为常数使用
5. 乘方请使用`**`符号，如：`3**2 = 9`

e.g.

函数
```
A*sin(x*y)
```

变量
```
x, 1
y , 2
```

输出
$$\frac{\partial^{3}}{\partial x\partial y^{2}} \left(A \sin{\left (x y \right )}\right) = - A x \left(x y \cos{\left (x y \right )} + 2 \sin{\left (x y \right )}\right)$$
